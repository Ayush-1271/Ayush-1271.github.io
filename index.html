A New Approach for Image Security Enhancement Using
Ternary Logic Linear Feedback Shift Register
for Cryptographic Applications
Proposed a novel ternary-field (GF(3)) LFSR-based stream cipher for encrypting medical and sensitive images. Unlike binary LFSRs, the base-3 feedback mechanism exponentially expands the keyspace, resisting brute-force and statistical attacks. Validated across entropy, NPCR, UACI, and pixel-correlation metrics.
